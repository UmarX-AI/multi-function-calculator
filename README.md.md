# Console-Based Calculator System

*Author:* Muhammad Umar

## Features
- Arithmetic Operations (Add, Subtract, Multiply, Divide, Power, Remainder, Square Root, etc.)
- BMI Calculator (Supports kg/m and lbs/inch)
- Unit Converters (Length, Weight, Temperature, Data, Speed)
- Input Validation
- Time-Based Greetings

## Usage
1. Run multi-function-calculator.py in Python.
2. Follow the menu prompts to select operations or converters.
3. Enter numbers when prompted.
4. Enjoy!

## Notes
- Ensure valid numeric input.
- BMI calculator supports metric and imperial units.
